# FULL STACK STARTUP (Easy Mode)
To start EVERYTHING (Data Server, AI Server, and Frontend) all at once in the background with live logging, run this single command from your root `folkloreGPT-main` directory:

bash start_everything.sh


---
# MANUAL STARTUP (If you want to run things in separate terminals)

Terminal 1 (Data Server):
cd data
npm install
node server.js

Terminal 2 (AI Server):
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 server_v2.py

Terminal 3 (Frontend):
cd frontend
npm install
npm start
